# Nimbus Cloud Platform

Nimbus Cloud Platform is my hands-on DevOps and Cloud Engineering project built to understand how modern applications are deployed, managed, and automated in real-world environments.

In this project, I am implementing a microservices-based e-commerce application using Linux, Shell Scripting, AWS, Terraform, Ansible, and Infrastructure as Code practices. The goal is to learn not only how applications are deployed today, but also how deployments, server management, monitoring, troubleshooting, and infrastructure provisioning were handled before DevOps automation became common.

Through this project, I am practicing Linux administration, service management using systemd, networking concepts, infrastructure provisioning, configuration management, deployment automation, troubleshooting production-like issues, and managing multiple environments. The platform serves as a practical learning environment to strengthen my DevOps, Cloud, and Automation skills through hands-on implementation rather than theory alone.
